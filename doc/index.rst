Introdução
==========

Pyndorama é um jogo para motivar o aprendizado através de atividades lúdicas, onde o conteúdo pode ser produzido por professores e alunos.

Motivação
=========

Apesar de muitas escolas terem recursos de informática, estes recursos são parcamentes utilizados para veicular o conteúdo didático do currículo corrente.

Objetivo
========

Criar uma plataforma onde alunos e professores interajam para construir conteúdos didáticos motivantes e simultaneamente introduza alunos e professores no uso e programação de sistemas de informática.
